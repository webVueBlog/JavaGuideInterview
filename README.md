## 3+1驱动学习法

3+1驱动学习法是一种有效的学习方式，它由以下四个步骤组成：

三步走战略（3步）：首先，学生需要了解概念或知识点。其次，他们需要尝试应用这些概念或知识点。最后，他们需要反思并总结所学内容。

重复练习（1步）：在完成以上三个步骤后，学生需要进行重复练习，以确保他们真正掌握了所学内容。

这种学习方法的关键在于学生重视反思和总结的过程，而不仅仅是吸收和记忆知识。
